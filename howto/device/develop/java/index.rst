Using Java SDK to Develop 
=========================

The articles in this section shows how to perform the most frequently used device end development such as to connect to the cloud, transmit data, and use extended services such as NTP.

To help you quickly get started with the Java SDK, we've prepared sample codes in `<dir>/src/main/java/com/envisioniot/enos/iot_mqtt_sdk/sample/`, where `<dir>` is root directory where you placed the Java SDK.
